# Grove-Guardians-Web

This repository hosts the public GitHub Pages deployment of the
Grove Guardians Progressive Web Application (PWA) — a digital story-coding environment
for German elementary school pupils developed as part of a master's
thesis at the Technical University of Munich.

The source code and active development are maintained in the
[Grove-Guardians](https://github.com/ls1intum/Grove-Guardians)
repository. This repository contains only the deployed build
artifacts.
